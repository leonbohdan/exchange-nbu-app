## Project Setup

```sh
npm install
```
or
```sh
yarn
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
or
```sh
yarn dev
```

### Compile and Minify for Production

```sh
npm run build
```
or
```sh
yarn bulid
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
or
```sh
yarn lint
```
